medlink

Medlink is a web based hospital pharmacy management system that helps to manage the pharmacy operations of a hospital. The main users of the system are the pharmacists and the wards. The system allows the pharmacists to manage the stock of the pharmacy, issue drugs to the wards, receive drugs from suppliers, and generate reports. Also the system allows the wards to request drugs from the pharmacy, receive drugs from the pharmacy, issue drugs to the patients and generate reports.

Technologies: React, Node js, Express, MongoDB, Figma
